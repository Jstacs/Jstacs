**NarrowPeakConvertToPromoter** Converts the narrowPeak file to promoter search.

The input of *NarrowPeakConvertToPromoter* is a narrowPeak file and the promoter sequences in FastA format with headers like "> *id* *chromosomeName*:*start*-*end*:*strand*", e.g. "> Os01g01010.1 Chr1:2602-3102:+"

If you experience problems using *NarrowPeakConvertToPromoter*, please contact_ us.
